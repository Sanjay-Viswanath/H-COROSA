Codes, MAT Files and Scripts for results presented in
Sanjay Viswanath, Manu Ghulyani, Muthuvel Arigovindan, "Structurally Adaptive 
Multi-Derivative Regularization for Image Recovery from Sparse Fourier Samples"
https://arxiv.org/abs/2105.12775
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

Run View_Table<x> to see Table <x> in paper as an array.

Press 'y' at the prompt to perform all reconstructions again to generate the
table.
Note that generating the results can take hours, depending on the system.

Press 'n' at the prompt to generate tables using reconstructions already stored in 
'Generated_Reconstruction_Results' folder.
