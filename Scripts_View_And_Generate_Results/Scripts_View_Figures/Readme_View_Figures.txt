Codes, MAT Files and Scripts for results presented in
Sanjay Viswanath, Manu Ghulyani, Muthuvel Arigovindan, "Structurally Adaptive 
Multi-Derivative Regularization for Image Recovery from Sparse Fourier Samples"
https://arxiv.org/abs/2105.12775
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

Run Figure_<x>_<y> to see Figure <x> in paper with description <y>.

For Figures with subfigures, displayed titles give the details including subfigure labels like (a),(b) etc.

Press 'y' at the prompt to perform reconstructions required to generate figures. 
Note that generating the results can take an hour or more, depending on the system.

Press 'n' at the prompt to generate figures using reconstructions already stored in 
'Generated_Reconstruction_Results' folder.
